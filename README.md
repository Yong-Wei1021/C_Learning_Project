# C Learning Project

This repository documents my journey of learning C programming and Linux system programming.

---

## 📁 Project Structure

### 01_basics

Fundamental C programming concepts and practice:

* pointer (pointer operations, pointer-to-pointer)
* memory (stack vs heap, malloc/free)
* bitwise operations
* linked list (basic implementation)

---

### 02_system_programming

Linux system-level programming practice:

* socket (TCP/IP communication)
* fork (process creation)
* pthread (multi-threading)

### 03_projects

A multi-client chat server built using C, socket, and pthread.

Features (planned / implemented):

* TCP socket communication
* Multi-client handling
* Multi-threading (pthread)
* Message broadcasting
